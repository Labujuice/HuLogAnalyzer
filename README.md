# PX4 ULog Analyzer 🛸 飛行日誌視覺化分析儀表板

> **極致流暢、無須安裝、資料不外傳的專業無人機飛行日誌分析工具。**

本專案是一個 100% 純前端（Client-side）的無人機 ULog 日誌分析儀表板。所有日誌解析與渲染均在本地瀏覽器（Web Worker & WASM）中完成，保證飛行數據的絕對隱私與安全。

### 🚀 [點此開啟線上分析儀表板 (Live App Demo)](https://labujuice.github.io/HuLogAnalyzer/dist/)

---

## 🌟 核心功能特色

### 📊 1. 高效能時序數據圖表 (uPlot)
* **大數據降採樣對齊 (LTTB)**：在 ULog 數據量極大時，採用 LTTB 演算法對時間軸及各欄位同步降採樣，徹底解決數據點數不一導致的縮線 Bug。
* **高流暢互動**：支援滑鼠滾輪縮放、中鍵拖曳平移、方向性左鍵拖曳（左拉放大、右拉縮小）、快速鍵一鍵重置視角（`Esc` / `R`）。
* **實時數值懸浮框**：完全繞過 React 渲染流程，直接以 DOM 驅動的數據 Tooltip，保證 60 FPS 級別的懸浮對齊流暢度。
* **多維度欄位拖曳**：支援 `Ctrl+Click` 與 `Shift+Click` 複選欄位，可成批拖曳進圖表中自動對齊並插值。

### 🛸 2. 3D 實時姿態與軌跡觀測器 (Three.js)
* **三維模型動態回放**：以 Three.js 動態建立高質感四軸飛行器模型，播放時實時解算四元數姿態進行三維旋轉。
* **起飛點為原點**：讀取日誌本地位置第一點作為相對原點 `(0, 0, 0)`，自動將 NED (北-東-地) 座標平移映射至 WebGL 空間。
* **歷史/未來雙線軌跡**：
  * **🔴 紅色實線**：代表已經飛過的歷史航線。
  * **🟡 黃色實線**：代表即將飛行的未來預計航線。
* **跟隨相機**：鏡頭焦點鎖定無人機平滑平移，並支援按住左鍵任意旋轉視角與滾輪縮放。

### ✈️ 3. AHRS 航空水平儀 HUD (Canvas 2D)
* **主飛行指示器**：在圓形遮罩下以 HTML5 Canvas 實時渲染航空級水平儀儀表。
* **刻度精準顯示**：包含動態傾斜指針、跟隨 Pitch 俯仰平移的俯仰天梯（刻度標記與朝向角），以及實時 Roll & Pitch 數位面板。

### 🗺️ 4. 2D GIS 地圖軌跡追蹤 (Leaflet)
* **無須 API Key 的底圖**：整合 Google 的衛星混合圖 (Satellite Hybrid) 與向量道路圖 (Roadmap)，隨時一鍵切換。
* **雙色路徑劃分**：
  * **🔴 紅色實線**：代表已飛過的歷史軌跡。
  * **🟡 黃色虛線**：代表尚未飛過的未來預計軌跡。
* **自動追蹤與鎖定**：具備「鎖定跟隨」與「完整視野縮放」切換，指針圖標根據偏航角（Yaw）實時旋轉。

### ⚡ 5. 極致流暢時脈與零 React 負載
* **時間發佈器機制 (`timePublisher`)**：建立純 JS 的高頻時間事件訂閱機制。播放時直接驅動 DOM 元素、uPlot、Three.js 與 Leaflet，**完全跳過 React 的 Virtual DOM 比對**，大幅降低 CPU 負載。
* **高精準時脈**：修正 React 渲染延遲對播放 RAF 迴圈的拖曳現象，播放速度精準契合物理時間並支援 0.25x ~ 10x 倍速。

## 🛠️ 本地開發與編譯

### 1. 安裝環境
本專案基於 **Node.js**、**Vite** 與 **TypeScript** 建置。請先複製專案並安裝依賴：
```bash
# 安裝相依套件 (包括 Three.js、uPlot、Leaflet 等)
npm install
```

### 2. 啟動本機開發伺服器
```bash
npm run dev
```
啟動後在瀏覽器開啟 `http://localhost:5173/` 即可進行開發與測試。

### 3. 編譯為生產環境靜態網頁
```bash
npm run build
```
編譯完成後，所有的靜態網頁產物均會輸出至 **`dist/`** 資料夾。
* 由於配置了相對路徑 `base: './'`，您可以直接雙擊 `dist/index.html` 進行本機離線開啟，或者直接將 `dist/` 上傳至 GitHub Pages 或任何靜態主機。
